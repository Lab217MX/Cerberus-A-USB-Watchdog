# Aislador USB ADUM3160

Módulo aislador USB basado en el chip ADUM3160. Proporciona aislamiento galvánico entre tu computadora y el dispositivo USB conectado.

## Protección contra

- Picos de voltaje
- Ruido eléctrico
- Diferencias de potencial de tierra
- Daños por cortocircuitos

## Archivos

### 1-Schematic_ADUM3160 USB Isolator.json

Esquemático del circuito.

- **Formato:** EasyEDA (`.json`)
- **Cómo abrir:** En EasyEDA, ir a `Archivo → Abrir → EasyEDA...` y seleccionar el archivo

### 1-PCB_USB_Isolator.json

Diseño del PCB.

- **Formato:** EasyEDA (`.json`)
- **Cómo abrir:** En EasyEDA, ir a `Archivo → Abrir → EasyEDA...` y seleccionar el archivo

## Notas

- Velocidad soportada: USB 2.0 Full Speed (12 Mbps)
- Puede usarse como complemento del proyecto Cerberus para añadir una capa extra de protección
